# 사업의 즐거움 — 북토크

최철용 저 『사업의 즐거움』 핵심 키워드 북토크 슬라이드.

🔗 **Live:** [joy.hansai.kr](https://joy.hansai.kr)

## 구성

- **1부 · 온라인 쇼핑몰 이야기** — 답안지/메타/네이버/쿠팡/아마존 (9개 슬라이드)
- **2부 · 사업 이야기** — 팀/숫자/시간/인정/자신을 믿는 것 (13개 슬라이드)
- **결론** — 사업의 즐거움 = 흑자 + 태도 + 관계 + 의미

## 조작법

- `→` / `Space` / `PageDown` — 다음 슬라이드
- `←` / `PageUp` — 이전 슬라이드
- `Home` / `End` — 처음 / 끝
- 화면 클릭 — 다음 슬라이드 (왼쪽 20%는 이전)

## 기술

순수 HTML/CSS/JS · Pretendard 폰트 · Vercel 정적 배포.
